eSubListRefresh
SINTAXIS
eSubListRefresh(callSrv/filePHP, idSubList, condition);
DESCRIPCION
Función para refrescar el contenido de una SubLista desde el servidor.
PARAMETROS
callSrv Nombre del [CallSrv] donde se ejecuta el sql o se rellena $usuCursor . También puede ser el nombre de un fichero php
idSubList Nombre de la SubLista donde se va a refrescar
condition Array hash con pares de valor "campo: valor", con los datos de la condición del sql 
EJEMPLO
[AddButton] * | Recargar SubList | | eSubListRefresh('RefreshSubList', '__ts_respuesta', {'cd_respuesta':32} );

[CallSrv] RefreshSubList
     $sql = "select ... where cd_respuesta={$cd_respuesta} ...";
     DB::query($sql);
Refresca la SubLista con los registros devueltos por el select
